Popup Template Designs

The Popup is rich with upstream and downstream dependencies. I have seen these dependencies as opportunities to establish working relationships with teams across multiple functionalities.

The Goal

Redesign the map popup to be mobile-first, easy to customize, and delightful to use. Design the user interface to be attractive yet discreet enough to complement a wide range of UI environments.

Status: Slated for October 2015 Beta Release

Skills Used

  • UI Design
  • Cross-Functional Collaboration
  • Rapid Prototyping
  • Usability Testing

UX / UI

Partnering with the UX architect and provided wireframes, I helped move the project forward by iterating through high-fidelity mockups. In order to ensure brand alignment and platform unity, I implemented Calcite colors, patterns, and fonts. I also reached out regularly to the icon team for guidance on consistent icon use.

As we socialized the mockups to cross-functional stakeholders, I was able to gather actionable feedback and improve my understanding of the broader implications of a new popup design.

Cross-Functionality

As the design achieved sufficient stakeholder approval and began to move into development, I continued to help advance the project by providing the development team with implementable stylesheets and HTML markup.

I ensured the efficient integration of my contributions by using the team's preferred collaboration tools and procedures. Supporting the development team in this way has fostered healthy cross-team integration.

Rapid Prototyping

Using my cross-functional integration, I was able to support a usability study on the popup. I created a prototype to simulate a working environment thus helping to ensure reliable feedback from users.

Icons for the new docking functionality.

Icons

The redesign introduces a new docking feature to the popup. Reaching out to the icon team, I discovered that there were no existing icons to represent docking functionality.

In order to facilitate the icon creation process, I provided the icon team with design concepts, documentation on the functionality associated with each icon, and examples of the UI environments in which the icons would be used.

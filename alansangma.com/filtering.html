App Template Filtering

My involvement in this project began with the desire to get reliable user feedback. My involvement grew as we incorporated user feedback into the design. And it has remained as we move toward implementation.

The Goal

Redesign the interface to better facilitate app template browsing and selection.

Status: Slated for October 2015 Release

Skills Used

  • UI Design
  • Cross-Functional Collaboration
  • Rapid Prototyping
  • Usability Testing

Usability to UI

As User Conference 2015 approached, I reviewed a spreadsheet that was intended to be a usability testing tool at UC. The goal was to learn the effectiveness of a proposed set of app template categories. With a desire to facilitate gathering the most accurate data, I took the initiative to rapidly create a touch-friendly interface that allowed facilitators to administer the test using tablet devices.

Administering the test with a clean, touch-friendly interface accelerated the testing process, improved the quality of the feedback, and deepened brand integrity.

With the usability documentation, I was able to quickly advance the design. I regularly sought out design feedback from cross-functional team members and employed the usability documentation as a guide to keep the user at the center of each design decision.

As development prepares for implementation, I have supported the development team by QA-ing implementation and by delivering usable code for styling and animations.

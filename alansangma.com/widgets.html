Calcite Map Widgets

The Map Widgets are integral tools used on nearly every map. Applying Calcite design standards to each of the many widgets is an essential part of bringing brand alignment and aesthetic unity to the platform.

The Goal

Restyle each map widget to align with Calcite design standards.

Status: Slated for December 2015 Release
Skills Used
  • UI Design
  • Cross-Functional Collaboration

Cross-Functional Coordination

I have taken a leadership role in this project by coordinating regular meetings with cross-functional team members. This means that while I am responsible for producing designs for each of the map widgets, I also facilitate the design review process. I use this dual role to practice the objective critique of my own designs and to build rapport with the development team.

Design and Implementation

I see this project as more than re-skinning the user interface. I see it as an opportunity to rethink and improve the user interface. At times, this has meant proposing significant changes to the UI. In these cases, I coordinated discussions with stakeholders (both local and remote) in order to gain crucial design feedback. I also used these discussions to elicit any concerns about implementing my proposed changes.
